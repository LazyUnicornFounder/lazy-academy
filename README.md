## Features

- **AI-generated personalized curriculum**
- **30-day learning plans**
- **Interactive quizzes**
- **Hands-on activities**
- **Progress tracking**

## Tech Stack

- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL, Auth, Edge Functions, Storage)
- **AI:** Claude / Gemini via API
- **Hosting:** Lovable
- **Build Tool:** Vite

## Getting Started

```bash
# Clone the repo
git clone https://github.com/LazyUnicornFounder/lazy-academy.git
cd lazy-academy

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Add your Supabase credentials to .env

# Start the dev server
npm run dev
```
